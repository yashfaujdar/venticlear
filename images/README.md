# Image Requirements for Venticlear Website

## Required Images

Please add the following images to this folder for the website to display properly:

### About Section
- **about-image.jpg** (1200x800px recommended)
  - Should show: Manufacturing facility, cleanroom, or air filtration systems in action
  - Professional, high-quality industrial photography

### Product Images (800x600px recommended)
All product images should be professional, high-resolution photos with clean backgrounds:

1. **pre-filters.jpg** - Pre-filter units
2. **fine-filters.jpg** - Fine filter systems
3. **mini-pleat.jpg** - Mini pleat filter modules
4. **hepa-filters.jpg** - HEPA filter units
5. **bag-filters.jpg** - Bag filter systems
6. **filter-modules.jpg** - Complete filter module assemblies
7. **bibo-systems.jpg** - BIBO (Bag-in/Bag-out) containment systems

## Temporary Placeholder Solution

Until you add your actual product images, the website will use placeholder images from Unsplash. The current placeholders are:
- Industrial/manufacturing themed images
- Air filtration and cleanroom environments
- Professional quality stock photos

## Image Specifications

- **Format**: JPG or PNG
- **Quality**: High resolution (at least 1920px width for hero images)
- **Optimization**: Compress images to keep file sizes under 500KB for faster loading
- **Aspect Ratios**: 
  - Product images: 4:3 (800x600px)
  - About image: 3:2 (1200x800px)
  - Hero background: 16:9 (1920x1080px)

## How to Add Images

1. Save your images with the exact filenames listed above
2. Place them in this `/images/` folder
3. The website will automatically load them

## Image Sources

You can source images from:
- Your own product photography
- Professional photographer
- Stock photo websites (ensure proper licensing):
  - Unsplash.com
  - Pexels.com
  - Shutterstock.com (paid)
  - iStock (paid)

## Tips for Best Results

- Use consistent lighting and backgrounds across product images
- Show products from multiple angles if possible
- Include scale references where helpful
- Ensure images are sharp and well-focused
- Use neutral or industrial backgrounds
- Consider adding lifestyle shots showing products in use
